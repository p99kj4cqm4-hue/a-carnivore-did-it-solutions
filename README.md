
# Section Solutions App

A simple React + Vite app for browsing 20 sections of 100 problems each and revealing each problem's solution.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push this folder to GitHub, then import the repo into Vercel.
Vercel should auto-detect Vite. If needed:

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
